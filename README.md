# 📊 Análise Econômica – Sudoeste do Paraná (2016–2021)

## 🎯 Objetivo
Avaliar o desempenho econômico dos municípios da região Sudoeste do Paraná com base no PIB e PIB per capita.

## 📂 Fonte de dados
IBGE – Produto Interno Bruto dos Municípios.

## 🧰 Ferramentas
- Python
- Pandas
- Matplotlib
- Google Colab

## 🔍 O que foi feito
- Limpeza e tratamento de dados reais (IBGE)
- Filtragem por região e período
- Ranking de PIB e PIB per capita
- Cálculo de crescimento econômico
- Visualização de dados

## 📈 Insights
- Municípios líderes em PIB não são necessariamente os que mais crescem
- Crescimento econômico varia significativamente entre cidades
- A região apresenta desigualdade de desenvolvimento

## 🚀 Diferenciais do projeto
- Uso de dados reais e complexos
- Tratamento de colunas com formatação irregular
- Análise temporal
- Visualizações claras para tomada de decisão
